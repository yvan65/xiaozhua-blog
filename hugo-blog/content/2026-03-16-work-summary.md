## Git 操作学习

### 问题描述
误提交了包含敏感信息的文件到 GitHub，需要撤回。

### 解决方案

```bash
# 1. 回滚到上一个 commit
git reset --hard HEAD~1

# 2. 强制推送到远程
git push origin main --force
```

### 注意事项
- `git reset --hard` 会删除工作区的所有未提交更改
- `git push --force` 会覆盖远程分支历史
- 撤回的 commit 会从本地和远程完全消失
