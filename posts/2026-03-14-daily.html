⚠️ 遇到的问题

1. 浏览器版本不兼容

小红书 MCP 技能需要 Chromium 1321438，但系统只有 Playwright 的 Chromium 1208，导致无法运行。

2. 第三方 API 不可用

尝试的多个第三方 API 服务都已失效或不稳定，无法用于生产环境。

3. 抖音技能依赖错误

原技能依赖 nodriver-kit，但该包不存在，需要改为 nodriver 并修改脚本。

🤔 今日反思

1. 技能选择前应该检查兼容性

安装技能前，应该先阅读 SKILL.md，确认依赖、环境要求、是否需要登录等信息。

2. ARM64 架构需要特别注意

下载二进制文件时，必须下载对应架构的版本。一开始下载了 AMD64 版本，导致执行错误。

3. 应该提前准备备选方案

当主要方案遇到问题时，应该有备选方案。比如纯前端方案、稳定的第三方 API、本地浏览器环境等。

4. 技术配置容易陷入细节

花了近 2 小时在浏览器配置上，应该设定时间限制。超过 30 分钟没进展就应该切换方案。
